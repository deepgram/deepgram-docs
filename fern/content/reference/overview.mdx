Welcome to the Deepgram API Reference! Here, you will find all the information you need to effectively integrate Deepgram's advanced speech recognition and transcription capabilities into your applications.

## Getting Started

To begin using the Deepgram API, check out our [Quick Start Guide](quick-start-guide). This guide will walk you through the initial steps of setting up your account, obtaining your API key, and making your first API call.

## API Endpoints

Deepgram offers a variety of API endpoints to cater to different needs:

- **Voice Agent API**: Build voice agents with our Voice Agent API.<br />Learn more in our [Voice Agent API documentation](/api/voice-agent/agent).
- **Speech-to-Text**: Convert audio files or streams into text.<br />Learn more in our [Speech-to-Text API documentation](/api/speech-to-text/listen).
- **Text-to-Speech**: Generate speech from text using our advanced models.<br />Explore the [Text-to-Speech API documentation](/api/text-to-speech/speak).
- **Management API**: Manage your projects, keys, and usage.<br />Detailed information can be found in the [Management API documentation](/api/management/models/list).

## Authentication

All API requests require authentication. You can authenticate using an API key or a JWT token. For more details, refer to our [Authentication Guide](authentication.mdx).

## Error Handling

Understanding how to handle errors is crucial for robust application development. Our [Error Handling Guide](errors.mdx) provides detailed information on common errors and how to resolve them.

## Rate Limits

Deepgram imposes rate limits on API requests to ensure fair usage and maintain service quality. For more details, refer to our [Rate Limits documentation](rate-limits.mdx).

## Support

If you need further assistance, our [Support Page](support) offers various resources, including FAQs, troubleshooting tips, and contact information for our support team.

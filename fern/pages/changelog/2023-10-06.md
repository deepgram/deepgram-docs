On **Oct 18th 2023 **Deepgram will be introducing a 90 day limit on the storage of log usage data for all of our customers.Â 

**What this means:**

- In the[ Deepgram Console](https://console.deepgram.com/login) > Usage > Logs users will now only be able to query for log data for up to 90 days of time.
- When using the Deepgram API [Usage](https://developers.deepgram.com/reference/get-all-requests) endpoint users will now only be able to query for usage data up to 90 days of time.
- You can still retrieve [Summarized Usage](https://developers.deepgram.com/reference/summarize-usage) for greater than 90 days.
